# АМЛ-таск
## О проекте

АМЛ-таск — это внутренний инструмент для управления задачами и проектами, объединяющий структурированный подход Jira (проекты, бэклог, спринты) и гибкость Trello (Kanban-доска с drag-n-drop).

Проект создается как удобное и интуитивное решение для команд, которым нужна прозрачность процессов без излишней сложности.


## Цель

Создать простой и быстрый инструмент, позволяющий:

- управлять задачами в рамках проектов
- планировать работу через бэклог и спринты
- отслеживать прогресс на Kanban-доске
- минимизировать порог входа для пользователей
